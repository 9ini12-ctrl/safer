# منصة إدارة السفراء (Static MVP)

منصة خفيفة تعمل على **GitHub + Netlify** بدون سيرفر (Static)، تشمل:

- رفع ملفات CSV بشكل مركزي:
  - **السفراء**: `name, phone, referral_code`
  - **التبرعات**: `amount, donor_phone, donation_date, referral_code`
  - **الصناديق**: `phone, box_id, created_at`
- حساب تحصيل السفير عبر **كود الإحالة**.
- حساب عدد الصناديق التي فتحها السفير عبر **رقم جواله**.
- صفحة سفير تُفتح عبر إدخال رقم الجوال.
- أهداف يومية مركزية (تحصيل + صناديق).
- رسالة نشر مركزية (نص + صورة) تظهر لكل سفير.

> ملاحظة مهمة: هذه النسخة تحفظ البيانات داخل المتصفح (LocalStorage).  
> إذا تحتاج "مركزي حقيقي" بين عدة أجهزة/سفراء — أضف تخزين سحابي (Netlify Functions / Supabase / Firebase).

---

## التشغيل محليًا

افتح ملف `index.html` مباشرة في المتصفح، أو استخدم خادم بسيط:

```bash
# Python
python -m http.server 8080
```

ثم افتح: `http://localhost:8080`

---

## النشر على Netlify

1) ارفع المشروع إلى GitHub.  
2) في Netlify:
- **New site from Git**
- اختر الريبو
- Build command: **(فارغ)**
- Publish directory: **/**

يوجد ملف `netlify.toml` لتفعيل إعادة التوجيه (SPA) إن احتجتها.

---

## لوحة الإدارة

- من أعلى الصفحة اضغط **لوحة الإدارة**
- كلمة المرور الافتراضية: `admin123`
- ارفع ملفات CSV ثم اضغط "حفظ البيانات"
- اضبط الأهداف ورسالة النشر والصورة
